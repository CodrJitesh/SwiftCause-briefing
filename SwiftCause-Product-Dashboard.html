Product Definition

SwiftCause is software on any device + branded accessories. It is NOT a kiosk. Supports both card-tap and QR/NFC flows.

Core Flow
Card Tap → QR / NFC → Magic Link → Gift Aid Declaration → Consent Ledger
30-day magic link token expiry. Consent Ledger Option 2 approved. Magic link shipped (PR #622).
Entity Model

Location

  • Immutable snapshots (HMRC audit)
  • Server-side at action time
  • Never a live reference

Kiosk (Collection Point)

  • Any device with SwiftCause app
  • Android Enterprise (Device Owner)
  • NFC HMAC for location proof

Campaign

  • Donation vs Activity mode
  • Compliance lock per campaign
  • Record type follows campaign
